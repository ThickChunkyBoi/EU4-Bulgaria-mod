How to install:
1. Launch the EU4 launcher
2. Go to Mod Library > Upload Mod > Create Mod
3. Enter a name, some tags, the directory (best if its the same name as the mod) and version (the current version can be seen in home page)
4. In "Documents\Paradox Interactive\Europa Universalis IV\mod" the launcher creates a new folder with a descriptor file and a descriptor file outside.
5. Paste the repository files (except the descriptor) inside the folder.
6. Make sure both descriptors have the same version, tags.
7. If the repository descriptor replaces files - copy paste them inside both of the descriptor files in your PC.
8. Enjoy
